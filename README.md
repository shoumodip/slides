# Slides
Simple Slideshow Application

## Quick Start
Depends on [Raylib](https://github.com/raysan5/raylib)

```console
$ ./build.sh
$ ./slides example
```

Use Vim keys or Arrow keys to navigate between slides. Clicking on the right or
left portions of the window will switch to that direction

## Drag and Drop
```console
$ ./slides
```

Drag and Drop any slideshow file (eg: `example`) into the application to load
it.
